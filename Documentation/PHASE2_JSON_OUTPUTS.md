# Phase 2: JSON Output Files

## Overview

All Phase 2 modules now automatically save their outputs to JSON files in a folder named after the video file. The output directory structure is:

```
data/
└── {video_name}/
    ├── video_metadata.json
    ├── frames_info.json
    ├── audio_metadata.json
    ├── audio_chunks_info.json
    ├── transcription.json
    ├── speaker_diarization.json
    ├── enhanced_transcript.json
    ├── speaker_statistics.json
    ├── slide_detection.json
    └── visual_embeddings_info.json
```

## Output Files

### 1. Video Preprocessing Module

#### `video_metadata.json`
- **Generated by**: `VideoProcessor.save_metadata_to_json()`
- **Contains**: Video file metadata (duration, FPS, resolution, codec, file size, etc.)
- **Example**:
  ```json
  {
    "file_path": "path/to/video.mp4",
    "file_name": "video.mp4",
    "file_size": 104857600,
    "file_size_mb": 100.0,
    "duration": 3600.0,
    "duration_formatted": "1h 0m 0s",
    "fps": 30.0,
    "frame_count": 108000,
    "width": 1920,
    "height": 1080,
    "resolution": "1920x1080",
    "codec": "h264",
    "aspect_ratio": 1.7777777777777777
  }
  ```

#### `frames_info.json`
- **Generated by**: `VideoProcessor.save_frames_info_to_json()`
- **Contains**: Information about extracted frames (paths, timestamps, indices)
- **Example**:
  ```json
  {
    "video_path": "path/to/video.mp4",
    "total_frames": 1080,
    "target_fps": 1.0,
    "frames": [
      {
        "frame_path": "path/to/frame_000001_5.234s.jpg",
        "timestamp": 5.234,
        "frame_index": 0
      }
    ]
  }
  ```

### 2. Audio Extraction Module

#### `audio_metadata.json`
- **Generated by**: `AudioExtractor.save_metadata_to_json()`
- **Contains**: Audio file metadata (duration, sample rate, channels, codec, bit rate, etc.)
- **Example**:
  ```json
  {
    "file_path": "path/to/audio.wav",
    "file_name": "audio.wav",
    "file_size": 11520000,
    "file_size_mb": 10.98,
    "duration": 3600.0,
    "duration_formatted": "1h 0m 0s",
    "sample_rate": 16000,
    "channels": 1,
    "codec": "pcm_s16le",
    "bit_rate": 256000,
    "bit_rate_kbps": 256.0
  }
  ```

#### `audio_chunks_info.json`
- **Generated by**: `AudioExtractor.save_chunks_info_to_json()`
- **Contains**: Information about audio chunks (paths, start times, indices)
- **Example**:
  ```json
  {
    "video_path": "path/to/video.mp4",
    "total_chunks": 120,
    "chunk_duration": 30.0,
    "chunks": [
      {
        "chunk_path": "path/to/chunk_0000_0.00s_30.00s.wav",
        "start_time": 0.0,
        "chunk_index": 0
      }
    ]
  }
  ```

### 3. Speech-to-Text Module

#### `transcription.json`
- **Generated by**: `SpeechToText.save_transcription_to_json()`
- **Contains**: Full transcription with segments, timestamps, language detection
- **Example**:
  ```json
  {
    "text": "Full transcribed text...",
    "language": "en",
    "language_probability": 0.99,
    "segments": [
      {
        "id": 0,
        "text": "Hello, welcome to this presentation.",
        "start": 0.0,
        "end": 3.5,
        "no_speech_prob": 0.01,
        "words": [
          {
            "word": "Hello",
            "start": 0.0,
            "end": 0.5,
            "probability": 0.99
          }
        ]
      }
    ],
    "num_segments": 150,
    "duration": 3600.0
  }
  ```

### 4. Speaker Diarization Module

#### `speaker_diarization.json`
- **Generated by**: `SpeakerDiarization.save_diarization_to_json()`
- **Contains**: Speaker segments with timestamps
- **Example**:
  ```json
  {
    "video_path": "path/to/video.mp4",
    "total_segments": 45,
    "unique_speakers": 3,
    "segments": [
      {
        "speaker_id": "SPEAKER_00",
        "start_time": 0.0,
        "end_time": 15.5,
        "duration": 15.5
      }
    ]
  }
  ```

#### `enhanced_transcript.json`
- **Generated by**: `SpeakerDiarization.save_enhanced_transcript_to_json()`
- **Contains**: Transcript segments with speaker attribution
- **Example**:
  ```json
  {
    "video_path": "path/to/video.mp4",
    "total_segments": 150,
    "segments": [
      {
        "text": "Hello, welcome to this presentation.",
        "start_time": 0.0,
        "end_time": 3.5,
        "speaker_id": "SPEAKER_00",
        "speaker_confidence": 0.95,
        "words": []
      }
    ]
  }
  ```

#### `speaker_statistics.json`
- **Generated by**: `SpeakerDiarization.save_speaker_statistics_to_json()`
- **Contains**: Statistics about each speaker (speaking time, segment counts, percentages)
- **Example**:
  ```json
  {
    "total_speakers": 3,
    "total_segments": 45,
    "total_duration": 3600.0,
    "speakers": {
      "SPEAKER_00": {
        "speaker_id": "SPEAKER_00",
        "total_segments": 20,
        "total_duration": 1800.0,
        "average_segment_duration": 90.0,
        "percentage": 50.0
      }
    },
    "speakers_sorted": []
  }
  ```

### 5. Visual Processing Module

#### `slide_detection.json`
- **Generated by**: `FrameExtractor.save_slide_detection_to_json()`
- **Contains**: Detected slide transitions with timestamps and similarity scores
- **Example**:
  ```json
  {
    "video_path": "path/to/video.mp4",
    "total_transitions": 25,
    "transitions": [
      {
        "frame_index": 150,
        "timestamp": 150.0,
        "similarity": 0.75,
        "change_magnitude": 0.25
      }
    ]
  }
  ```

#### `visual_embeddings_info.json`
- **Generated by**: `VisualEmbeddings.save_embeddings_info_to_json()`
- **Contains**: Metadata about visual embeddings (not the actual embeddings, which are too large)
- **Example**:
  ```json
  {
    "video_path": "path/to/video.mp4",
    "model_name": "ViT-B/32",
    "num_frames": 1080,
    "embedding_dim": 512,
    "embeddings_shape": [1080, 512],
    "has_timestamps": true,
    "timestamps": [0.0, 1.0, 2.0, ...],
    "note": "Actual embeddings are not saved to JSON (too large). Use numpy.save() to save embeddings."
  }
  ```

## Usage Examples

### Example 1: Video Processing with JSON Output

```python
from src.preprocessing.video_processor import VideoProcessor

processor = VideoProcessor(fps=1.0)
video_path = "data/input/video.mp4"

# Extract frames
frames = processor.extract_frames(video_path)

# Get metadata
metadata = processor.get_video_metadata(video_path)

# Save to JSON
processor.save_metadata_to_json(video_path, metadata)
processor.save_frames_info_to_json(video_path, frames)
```

### Example 2: Audio Processing with JSON Output

```python
from src.preprocessing.audio_extractor import AudioExtractor

extractor = AudioExtractor()
video_path = "data/input/video.mp4"

# Extract audio
audio_path = extractor.extract_audio(video_path)

# Get metadata
metadata = extractor.get_audio_metadata(audio_path)

# Chunk audio
chunks = extractor.chunk_audio(audio_path)

# Save to JSON
extractor.save_metadata_to_json(video_path, audio_path, metadata)
extractor.save_chunks_info_to_json(video_path, chunks)
```

### Example 3: Transcription with JSON Output

```python
from src.audio.speech_to_text import SpeechToText

stt = SpeechToText()
video_path = "data/input/video.mp4"
audio_path = "data/input/audio.wav"

# Transcribe
transcription = stt.transcribe(audio_path)

# Save to JSON
stt.save_transcription_to_json(video_path, transcription)
```

### Example 4: Speaker Diarization with JSON Output

```python
from src.audio.speaker_diarization import SpeakerDiarization
from src.audio.speech_to_text import SpeechToText

diarization = SpeakerDiarization()
stt = SpeechToText()
video_path = "data/input/video.mp4"
audio_path = "data/input/audio.wav"

# Diarize
diarization_results = diarization.diarize(audio_path)

# Transcribe
transcription = stt.transcribe(audio_path)

# Assign speakers
enhanced_segments = diarization.assign_speakers_to_segments(
    transcription["segments"],
    diarization_results
)

# Get statistics
statistics = diarization.get_speaker_statistics(diarization_results)

# Save to JSON
diarization.save_diarization_to_json(video_path, diarization_results)
diarization.save_enhanced_transcript_to_json(video_path, enhanced_segments)
diarization.save_speaker_statistics_to_json(video_path, statistics)
```

### Example 5: Visual Processing with JSON Output

```python
from src.visual.frame_extractor import FrameExtractor
from src.visual.visual_embeddings import VisualEmbeddings

frame_extractor = FrameExtractor()
visual_emb = VisualEmbeddings()
video_path = "data/input/video.mp4"

# Extract frames
frames_with_timestamps = frame_extractor.extract_frames(video_path)
frames = [f[0] for f in frames_with_timestamps]
timestamps = [f[1] for f in frames_with_timestamps]

# Detect slides
slide_changes = frame_extractor.detect_slides(frames, timestamps)

# Extract embeddings
embeddings = visual_emb.extract_embeddings(frames)

# Save to JSON
frame_extractor.save_slide_detection_to_json(video_path, slide_changes)
visual_emb.save_embeddings_info_to_json(video_path, embeddings, timestamps)
```

## Output Directory Location

All JSON files are saved in:
```
data/{video_name}/
```

Where `{video_name}` is the stem (filename without extension) of the input video file.

For example, if the video is `data/input/presentation.mp4`, the JSON files will be saved in:
```
data/presentation/
```

## Notes

1. **Visual Embeddings**: The actual embedding arrays are not saved to JSON (they're too large). Only metadata is saved. To save embeddings, use `numpy.save()`:
   ```python
   import numpy as np
   np.save("data/presentation/visual_embeddings.npy", embeddings)
   ```

2. **Automatic Directory Creation**: The output directory is automatically created if it doesn't exist.

3. **JSON Formatting**: All JSON files are formatted with 2-space indentation for readability.

4. **Error Handling**: If saving fails, errors are logged but don't stop processing.

5. **File Naming**: JSON file names are standardized and consistent across all modules.

## Integration with Processing Pipeline

When implementing the full processing pipeline, you can call these save methods after each processing step:

```python
# 1. Video preprocessing
frames = processor.extract_frames(video_path)
metadata = processor.get_video_metadata(video_path)
processor.save_metadata_to_json(video_path, metadata)
processor.save_frames_info_to_json(video_path, frames)

# 2. Audio extraction
audio_path = extractor.extract_audio(video_path)
audio_metadata = extractor.get_audio_metadata(audio_path)
chunks = extractor.chunk_audio(audio_path)
extractor.save_metadata_to_json(video_path, audio_path, audio_metadata)
extractor.save_chunks_info_to_json(video_path, chunks)

# 3. Transcription
transcription = stt.transcribe(audio_path)
stt.save_transcription_to_json(video_path, transcription)

# 4. Speaker diarization
diarization_results = diarization.diarize(audio_path)
enhanced_segments = diarization.assign_speakers_to_segments(
    transcription["segments"], diarization_results
)
statistics = diarization.get_speaker_statistics(diarization_results)
diarization.save_diarization_to_json(video_path, diarization_results)
diarization.save_enhanced_transcript_to_json(video_path, enhanced_segments)
diarization.save_speaker_statistics_to_json(video_path, statistics)

# 5. Visual processing
frames_with_timestamps = frame_extractor.extract_frames(video_path)
frames = [f[0] for f in frames_with_timestamps]
timestamps = [f[1] for f in frames_with_timestamps]
slide_changes = frame_extractor.detect_slides(frames, timestamps)
embeddings = visual_emb.extract_embeddings(frames)
frame_extractor.save_slide_detection_to_json(video_path, slide_changes)
visual_emb.save_embeddings_info_to_json(video_path, embeddings, timestamps)
```
